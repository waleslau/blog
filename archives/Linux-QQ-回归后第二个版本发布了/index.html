Linux平台的桌面用户虽然少，但你们要么不做，要么就好好好做。腾讯一向不开放微信/QQ的API给第三方（不造为啥，好像全世界的大型 IM 都在搞封闭），甚至隔三差五的去封杀一些开源的实现。如果腾讯不如此旗帜鲜明地反对第三方客户端，凭QQ的用户量，开源社区早就能把QQ For Linux做出来。而不是现在这个可以说是在糊弄人的 QQ For Linux (对开发者没有恶意，即使这个版本，也比 wine 方案稳定得多了，就是功能少了点，总之还是挺香的)。

这几年政府大力推行国产系统，或许未来某个时间，上面有需要了，腾讯就会在 Linux 平台增加一些资源投入吧
